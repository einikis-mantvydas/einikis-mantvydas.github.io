Project Interests

Package Management

I am highly interested in System Administration. Troubleshooting, repairing and keeping up to date with the newest systems. Currently working on a project for package management. Package manager systems help automate tasks of administering software. Package manager systems manage dependencies, and it allows and provides methods of installing and managing those said dependencies. It finds the correct package, and also displays different versions of the package, so the correct one can then be installed, as well as the package manager checks for vulnerabilities. In the IT Industry they are used to download and manage 3rd party dependencies securely.

Virtualization

I am also interested in virtualization. For example, if a small-medium sized business decides to virtualize their services (MS Exchange servers, Linux Development servers, Remote desktops for users)
This involves working with Type-1 (bare-metal) hypervisors. For example, a company has many different services on many different servers. This takes up space, and makes management of those services harder. To make it more manageable, the Type-1 hypervisor can host all of those services in the form of a virtual server.
