Accordion

Accordion is useful in this case, since displaying 12 rows of grades in years 1 and 2 takes a lot of space up. Accordion feature makes it nice and compact, if user click on "Year 1", a table with number, module and grade appears, as shown in Figure 4.
The "Year 4" Accordion is set to color "warning", as that year is in progress and that option is disabled. If a user tries to click on "Year 4", it will not do anything, as it is disabled.
